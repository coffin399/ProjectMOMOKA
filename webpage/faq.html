FAQ

MOMOKA（PLANA / ARONA）の使い方に関する質問です。動かないときは トラブルシューティング を先に確認してください。

Common questions about MOMOKA (PLANA / ARONA). If something is broken, start with Troubleshooting.

MOMOKA / PLANA / ARONA の関係は？ What are MOMOKA, PLANA, and ARONA?

答え: MOMOKA はプロジェクト名です。実際に招待する Discord ボットは PLANA（プライマリ）と ARONA（コンパニオン）の2体です。1プロセスで両方を動かしています。

Answer: MOMOKA is the project name. The Discord bots you invite are PLANA (primary) and ARONA (companion), run together in one process.

どっちを招待すればいい？ Which bot should I invite?

答え: まずは PLANA を招待してください。TTS・画像・通知・トラッカー・Link Fix・メディアダウンロードなどは PLANA のみです。LLM の討論（debate）やクロスチェックを使う場合は、同じサーバーに ARONA も 入れてください。

Answer: Start with PLANA. TTS, images, notifications, trackers, Link Fix, and media download are PLANA-only. For LLM debate / cross_check, invite ARONA as well to the same server.

必要な権限は？ What permissions are needed?

招待リンクに含まれる権限一式を付与してください。チャンネルごとに閲覧・送信・履歴・埋め込みリンク・ボイス接続などが拒否されていると動きません。不具合の多くは権限不足が原因です。詳しくは トラブルシューティング を参照してください。

Grant the permission set included in the invite link. Channel-level denies (view, send, history, embed links, voice, etc.) will break features. Most issues are permission-related — see Troubleshooting.

データはどう扱われる？ How is my data handled?

機能提供のため、メッセージ内容や ID などが処理・ログ・外部 API に渡ることがあります。詳細は プライバシーポリシー を読んでください。

Message content and IDs may be processed, logged, or sent to third-party APIs to provide features. See the Privacy Policy.

Link Fix を止めたい How do I stop Link Fix?

  • そのメッセージだけ: 本文に fxignore を含める、または URL を <...> で囲む
  • Per message: include fxignore, or wrap the URL in <...>
  • サーバー単位: チャットでスラッシュコマンド /linkfix を実行（サーバー管理権限が必要）。設定画面で機能全体・全サイト一括・サイト別の無効化（デフォルトは有効）
  • Per server: run the /linkfix slash command in Discord (Manage Server required). Disable via the settings panel — master, bulk-all, or per-site (enabled by default)

ARONA に TTS などを頼むと？ What if I ask ARONA for TTS / images / etc.?

ARONA は LLM・音楽・一部ユーティリティ中心です。TTS・画像・通知・トラッカー・Link Fix などは PLANA 専用のため、PLANA への誘導メッセージが返ります。

ARONA focuses on LLM, music, and some utilities. TTS, images, notifications, trackers, and Link Fix are PLANA-only — ARONA will redirect you to PLANA.

セルフホストはできる？ Can I self-host?

ソースは公開していますが、依存が多く上級者向けです。一般利用は公開ボットの招待を推奨します。詳細は GitHub を参照してください。

The source is public but heavy and aimed at advanced users. For normal use, invite the hosted bots. See GitHub.

動かないときは？ Something is not working?

トラブルシューティング を確認してください。多くの場合は権限不足か、ボットが一時オフラインです。

See Troubleshooting. Most issues are missing permissions or a temporary offline period.

Troubleshooting

まず疑うのは権限不足です。チャンネル権限や招待時のパーミッションを見直し、必要なら招待リンクから再付与してください。ボット全体がオフラインのときは、しばらく待ってから再度お試しください。

Start by checking missing permissions. Review channel permissions and the invite permission set; re-invite via the invite link if needed. If bots are offline site-wide, wait a while and try again.

ボットがオフラインのとき When the bot is offline

開発者の PC にアップデート等が入っていて、本人が気づいていないだけのことが多いです。しばらく待ってから再度お試しください。長時間続く場合のみサポートへご連絡ください。

Often the developer’s PC is updating (or similar) and they simply have not noticed yet. Please wait a while and try again. Contact support only if it stays offline for a long time.

ボットがオフライン / 全体的に応答しない Bot offline / no response at all

  1. 上記のとおり、しばらく待つ
  2. Wait for a while, as noted above
  3. 長時間続く場合は サポートサーバー、または coffin299 へ DM
  4. If it lasts a long time: support server, or DM coffin299

AI / メンションに反応しない AI / mentions do not respond

  • チャンネルで Bot の閲覧・メッセージ送信・履歴閲覧・アプリコマンド使用が許可されているか
  • Channel allows the bot to view, send, read history, use application commands
  • 正しいボット（PLANA または ARONA）をメンションしているか
  • You are mentioning the correct bot (PLANA or ARONA)
  • 解決しなければ招待リンクから再招待して権限を付け直す
  • If still broken, re-invite with the invite link to refresh permissions

スラッシュコマンドが出ない Slash commands missing

招待スコープや権限不足が原因であることが多いです。/invite やサイトの招待リンクから再招待し、反映まで少し待ってください。

Usually invite scope or permissions. Re-invite via /invite or the site invite links, then wait a bit for Discord to sync.

debate / cross_check が動かない debate / cross_check does not work

同じサーバーに PLANA と ARONA の両方が必要です。不足している方を /invite から追加してください。

Both PLANA and ARONA must be in the same server. Add the missing one via /invite.

ARONA で TTS・画像・通知などが使えない TTS / images / notifications fail on ARONA

仕様です。これらの機能は PLANA のみです。誘導に従って PLANA を使ってください。

By design — those features are PLANA-only. Follow the redirect and use PLANA.

音楽が鳴らない Music does not play

  • ボイスチャンネルへの接続・発言権限
  • Connect / Speak in the voice channel
  • Bot がその VC に入れるか、他 Bot との競合
  • The bot can join that VC; check conflicts with other bots
  • 権限を直してもダメならサポートへ
  • If permissions look fine, contact support

Link Fix が動かない Link Fix does not run

  • メッセージ送信・埋め込みリンク（元メッセージの embed 抑制にはメッセージ管理）
  • Send Messages / Embed Links (Manage Messages helps suppress broken embeds)
  • 公式 embed が既にある場合は発火しないことがあります
  • May not fire when Discord already shows a normal embed
  • fxignore / URL を <> 囲み / /linkfix で無効化していないか
  • Check fxignore, <>-wrapped URLs, or /linkfix disabled

地震・Twitch 通知が来ない Earthquake / Twitch notifications missing

PLANA で通知チャンネルを設定し、そのチャンネルへの投稿権限があるか確認してください。

Configure channels on PLANA and ensure the bot can post in that channel.

メディアダウンロードが失敗する Media download fails

チャンネル権限のほか、URL 非対応や運営側の一時制限のことがあります。繰り返す場合はサポートへ。

Besides channel permissions, the URL may be unsupported or temporarily limited. Contact support if it keeps failing.

画像検索が拒否される Image search is rejected

NSFW 系コマンドは Discord の NSFW チャンネルでのみ利用できます（安全版コマンドを除く）。

NSFW image commands require a Discord NSFW channel (except safe variants).

権限更新のヒントが出る Permission update hint appears

ボットが不足権限を検出しています。表示された招待リンクから権限を付け直してください。

The bot detected missing permissions. Use the shown invite link to re-grant them.

それでも解決しない Still stuck

  • Support server
  • Discord: coffin299（DM）

  • 可能ならサーバー ID と再現手順を添えてください
  • Include the server ID and steps to reproduce if you can

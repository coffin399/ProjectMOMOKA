プライバシーポリシー

Privacy Policy

最終更新: 2026-07-25。本ポリシーは、公開運用中の MOMOKA（PLANA / ARONA）が取り扱う情報について説明します。日英で差異がある場合は日本語を優先します。

Last updated: 2026-07-25. This policy describes information handled by the hosted MOMOKA bots (PLANA / ARONA). If Japanese and English differ, Japanese takes priority.

取得しうる情報 Information we may process

  • Discord のユーザー ID・表示名、ギルド ID、チャンネル ID、ロール情報（権限判定に必要な範囲）
  • Discord user IDs / display names, guild IDs, channel IDs, and role info as needed for permissions
  • メッセージ本文・添付（AI 応答、読み上げ、ログ等のため）
  • Message content and attachments (for AI replies, TTS, logging, etc.)
  • ボイスチャンネルの参加状態（音楽・TTS）
  • Voice channel state (music / TTS)
  • スラッシュコマンドの入力内容、ゲーム名タグ等の検索クエリ
  • Slash command inputs and search queries (e.g. game name tags)
  • サーバー管理者が設定した通知先・Link Fix・TTS 設定など
  • Admin-configured notification channels, Link Fix, TTS settings, etc.

サーバー側での保持 Server-side retention

ギルド設定などは主に運用マシン上の JSON 等に保存されます（例: TTS 設定、自動参加ユーザー ID、通知チャンネル、Link Fix 設定、チャンネルごとのモデル選択）。AI の会話履歴は原則としてメモリ上のみで、再起動で消えます（送信時は直近のやり取りを外部 API に渡します）。

Guild settings are mainly stored as JSON on the operator’s machine (TTS settings, auto-join user IDs, notification channels, Link Fix, per-channel model choice, etc.). Conversation history is generally in memory only and cleared on restart (recent turns are still sent to AI APIs while active).

ログ Logs

障害対応・不正防止のため、ギルド名・ユーザー名・ユーザー ID・入力の一部などが運用ログに残ることがあります。PLANA では、設定した Discord チャンネルへログが転送される場合があります。

For operations and abuse prevention, logs may include guild names, usernames, user IDs, and excerpts of input. PLANA may forward logs to configured Discord channels.

第三者への送信 Third-party transmission

機能に応じて、次のような外部サービスへデータが送られることがあります。

Depending on the feature, data may be sent to services such as:

  • LLM プロバイダ（会話テキスト・画像）
  • LLM providers (chat text / images)
  • Web 検索（検索クエリ）
  • Web search (queries)
  • 動画・音楽配信先（yt-dlp 経由の URL 等）
  • Media platforms via yt-dlp (URLs, etc.)
  • Twitch / 地震情報 API、Valorant / R6S API
  • Twitch / earthquake APIs, Valorant / R6S APIs
  • Link Fix 用プロキシ（書き換え後 URL の取得）
  • Link Fix proxy domains (fetching rewritten URLs)
  • Google Drive（メディアダウンロードの一時保管。共有リンク発行後、短時間で削除）
  • Google Drive (temporary media downloads; files are deleted after a short period)

利用目的 Purposes

機能提供、障害対応、不正防止、品質改善のために利用します。販売目的のプロファイリングは行いません。

We use data to provide features, fix issues, prevent abuse, and improve quality. We do not sell profiling data.

保持・削除の要望 Retention and deletion requests

設定データはギルドでボットを利用している間保持されることがあります。会話履歴は揮発的です。削除や確認の要望は、技術的に可能な範囲で対応します。サポートサーバー、または Discord の coffin299 へ DM してください。

Settings may be kept while the bot is used in a guild. Chat history is ephemeral. Deletion/access requests are handled where technically feasible via the support server, or DM coffin299 on Discord.

Theme and language preferences are stored in browser localStorage. We also use Google Analytics (measurement ID: G-46J394MX23) for traffic analysis; cookies and related data may be sent to Google. See Google’s Privacy Policy. The host may also collect access logs.
